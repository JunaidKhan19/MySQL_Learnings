Assignment –6
Using Special Operators in Conditions.

1) Write two different queries that would produce all orders taken on October 3rd or 4th, 1990.
query1:
select onum, odate, cnum, amt from orders 
where odate between '1990-10-03' and '1990-10-04';

query2:
select onum, odate, cnum, amt from orders 
where odate in ('1990-10-03', '1990-10-04');

2) Write a query that selects all of the customers serviced by Peel or Motika
(Hint: the snum field relates the two tables to one another).
main query:
select cnum , cname, city, rating, snum from customers
where snum in(1001, 1004);

query using joins:
select cs.cnum, cs.cname, cs.city, cs.rating, cs.snum, sp.sname, sp.city, sp.comm from customers cs, salespeople sp where sp.snum = cs.snum;

3) Write a query that will produce all the customers whose names begin with a letter from ‘A’ to ‘G’.
select cnum, cname, city, rating, snum
from customers
where cname >= 'A'
  and cname < 'H';

4) Write a query that selects all customers whose names begin with the letter ‘C’.
select cnum, cname, city, rating, snum
from customers
where cname like'C%';

5) Write a query that selects all orders except those with zeroes or NULLs in the amt field.
select onum, odate, cnum, amt from orders
where amt is not null;
